# SimpleBreathing

Application Android très simple de respiration, développée avec Godot 4.6.2 et C#.

## État actuel

- Écran principal minimal : jauge + boule + bouton démarrer + bouton réglages.
- Écran de réglages séparé.
- Durées d'inspiration et d'expiration modifiables.
- Thèmes de couleurs sélectionnables.
